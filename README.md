# Zomato Customer Analysis

## Objective
The goal of this project is to analyze the Zomato restaurant dataset to understand customer behavior, popular cuisines, rating patterns, and pricing trends.

##ью## Dataset
Public Zomato dataset containing 10,000+ restaurant records including location, cuisine type, ratings, cost and online delivery availability.

## Steps Performed
- Data Cleaning (handled missing values and duplicates)
- Feature understanding and preprocessing
- Exploratory Data Analysis (EDA)
- Statistical observations of ratings and pricing
- Visualization of trends using graphs and charts

## Key Insights
- Restaurants with rating > 4.2 attract significantly higher customer engagement
- North Indian and Chinese cuisines dominate demand
- Mid-range priced restaurants receive the highest ratings
- Online delivery availability increases popularity

## Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Seaborn, Jupyter Notebook

## Outcome
This project helps businesses understand customer preferences and optimize pricing and cuisine offerings for higher engagement.
